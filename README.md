# tes-javan-crud-desa

## Project setup
```
1. Clone This project
2. cd tes-javan-crud-desa (or directory of cloned this project)
3. create a database on pgsql
4. composer install
5. cp .env.example .env
6. set the environment variables (config database, app kee , etc) on .env file
7. php artisan key:generate
8. php artisan migrate
9. php artisan db:seed
10. import ke postman collection api dari file "Test Koding Javan 2.postman_collection.json"
11. import ke postman environment api dari file "Test Koding Javan CRUD Desa.postman_environment.json"
12. pastikan sudah menginstall laravel herd di mana url project ini adalah 
"http://test-javan-crud-desa.test"


```

### Run
```
1. buka postman
2. run api di dalam collection Test Koding Javan CRUD Desa

```
